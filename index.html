    Run Tracker

    This project was my first introduction to full stack development. The goal of the project was to create a website where people could log their run distances and times, and then be able to see all the runs they have logged. To accomplish such, I setup a database with a table to store user data and a table to store the run data. I used Ruby on Rails to set this up and then hosted the API on heroku. As I worked through the front end, I ran into many hurdles handing time as it's compatibility varied depending on browser so I had to create an alternate solution.
